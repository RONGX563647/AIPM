# AI全自动研发大脑 - 创新布局Figma设计提示词（黑金高级版）

## 项目基础信息
**项目名称**：AI全自动研发大脑  
**项目定位**：面向研发团队的下一代智能研发管理平台  
**设计理念**：融合极简主义、空间感和智能交互，打破传统B端布局模式

## 全局创新布局体系

### 1. 革命性导航系统
```
■ 智慧侧边栏（Smart Sidebar）：
  - 形态：30px超窄悬浮侧边栏（默认状态），hover展开为240px详细菜单
  - 材质：深空黑玻璃质感（背景：rgba(17,17,19,0.95)，backdrop-filter: blur(20px)）
  - 交互：悬停展开、点击固定、滑动隐藏
  - 内容：纯金色线性图标 + tooltip提示，展开后显示模块名称和快捷操作

■ 智能顶部栏（Intelligent Topbar）：
  - 分区设计：
    • 左侧：全局搜索（30%宽度，金色发光搜索框）
    • 中部：面包屑导航 + 上下文操作栏（动态变化）
    • 右侧：用户中心（头像 + 下拉菜单） + 全屏/设置/通知（悬浮按钮群）

■ 底部工作栏（Bottom Dock）：
  - 位置：底部10px悬浮，宽度80%，居中显示
  - 内容：常用工具（计算器、便签、截图）、系统状态（CPU/内存）、快捷操作（新建、刷新、保存）
  - 交互：鼠标移入上浮，移出下沉，始终可用但保持不干扰
```

### 2. 主工作区创新布局
```
核心原则：模块化、可定制、零干扰

■ 工作台模式（Dashboard Mode）：
  - 布局：无边界自由布局，模块可任意拖拽、缩放、组合
  - 模块容器：直角黑色卡片，仅左侧显示金色竖线标识类型
  - 间距：智能间距系统，模块靠近时自动对齐，远离时保持最小间距

■ 专注模式（Focus Mode）：
  - 激活方式：点击模块标题栏的「专注」按钮
  - 效果：当前模块全屏化，其他模块半透明后退，聚焦当前任务
  - 导航：顶部显示快速返回按钮和同类型模块切换箭头

■ 分屏模式（Split Mode）：
  - 支持：二分屏（左右/上下）、三分屏（左中右/上中下）
  - 操作：拖动模块到屏幕边缘自动吸附，显示分屏预览
  - 记忆：保存用户常用分屏组合
```

### 3. 空间层级系统
```
深度层级规划：
Level 0：背景层（#000000 纯黑，极轻微噪点纹理）
Level 1：内容层（主工作区，卡片悬浮高度1）
Level 2：侧边栏/顶部栏（悬浮高度2）
Level 3：模态框/抽屉（悬浮高度3）
Level 4：通知/提示（悬浮高度4）
Level 5：加载遮罩/全屏覆盖（悬浮高度5）
```

## 各页面创新设计详解

### 1. 登录页 - 「时空之门」
```
布局概念：中心扩散式设计
• 背景：深空动态粒子背景（缓慢运动的金色粒子）
• 中心：金色圆形登录面板（直径400px）
• 扩散环：三层金色圆环，象征不同权限层级
• 登录方式：用户名密码位于内环，第三方登录位于外环
• 交互动效：输入框获得焦点时，对应圆环亮度增强
• 登录按钮：位于最中心，圆形金色按钮，点击后产生波纹扩散效果
```

### 2. 首页（工作台） - 「智慧大脑」
```
布局概念：神经元网络式布局
• 核心：中央数据仪表盘（金色环形图表，显示系统健康度）
• 分支：6个主要功能模块作为神经元节点，放射状分布在核心周围
• 连接线：模块间用金色虚线连接，显示数据流动关系
• 交互：点击模块节点展开详情，连接线变为实线
• 动态效果：关键数据变化时，对应模块脉动发光

数据展示创新：
• 3D数据卡片：卡片轻微旋转，正面显示数据，侧面显示趋势
• 呼吸式指示器：异常数字缓慢闪烁红光，正常数字恒定金色
• 数据河流：底部显示实时数据流（部署日志、接口调用等）
```

### 3. 项目管理页 - 「宇宙星图」
```
布局概念：星际导航界面
• 背景：星图网格（极细的金色网格线，部分节点为星星）
• 项目表示：每个项目为一个星系（中心星球 + 环绕的卫星任务）
• 缩放导航：鼠标滚轮缩放，拉近查看项目详情，拉远查看整体关系
• 筛选系统：右上角星座图筛选器，不同星座代表不同状态/技术栈
• 时间维度：顶部时间轴，可查看项目在不同时间点的状态

项目卡片创新：
• 球形项目图标：可旋转查看不同面（正面：概览，侧面：技术栈，背面：成员）
• 引力场：相关项目互相靠近，无关项目自动远离
• 轨道系统：任务像卫星一样围绕项目旋转，轨道颜色表示优先级
```

### 4. 任务&进度管理页 - 「时间河流」
```
布局概念：时间线河流可视化
• 主视觉：水平时间河流，任务像船只一样在河流中航行
• 河流分段：上游（待办）、中游（进行中）、下游（已完成）、瀑布（阻塞）
• 任务卡片：船型卡片，大小表示优先级，吃水深度表示进度
• 交互：拖动任务在河流中移动，改变时间顺序和状态
• 瀑布视图：阻塞任务显示为瀑布跌落动画

甘特图创新：
• 3D甘特图：任务条有厚度，依赖关系用金色光缆连接
• 时间透镜：鼠标悬停区域时间拉伸，查看详细时间规划
• 进度地形图：下方显示进度地形，山峰表示进度快，山谷表示阻塞
```

### 5. 智能接口中心页 - 「神经网络」
```
布局概念：生物神经网络模拟
• 接口节点：每个接口为发光节点，不同颜色表示不同HTTP方法
• 连接网络：接口间的调用关系用金色脉冲光线连接
• 数据流：实时数据在连接线上流动，显示调用频率
• 集群视图：相关接口自动聚类，形成功能模块
• 调试模式：点击接口节点，右侧展开调试面板，连接线变粗显示数据流向

接口文档创新：
• 3D接口文档：接口参数、请求头、响应分层显示，可旋转查看
• 智能填充：根据历史请求自动填充测试数据
• 实时监控：调用时显示金色粒子沿着连接线流向目标节点
```

### 6. AI代码评审页 - 「代码宇宙」
```
布局概念：三维代码空间
• 左侧：代码编辑器作为「地面」，代码文字略微浮起
• 右侧：评审结果作为「天空」，问题点用星座图表示
• 交互：点击问题星座，编辑器对应行高亮，并显示连接光束
• 3D视图：可旋转视角查看代码结构（函数为山峰，类为建筑，循环为漩涡）

代码可视化创新：
• 热力图覆盖：代码复杂度、重复率用颜色半透明覆盖
• 依赖关系图：函数调用关系在3D空间中用金色光线连接
• 时间旅行：查看代码历史版本，像地质层一样叠加显示
• AI建议气泡：优化建议以气泡形式从代码中浮现，可拖拽采纳
```

### 7. 自动化接口测试页 - 「实验室」
```
布局概念：科幻实验室界面
• 测试台：中央为测试平台，用例像实验瓶一样排列
• 仪器面板：左侧为参数调节面板（滑块、旋钮、开关）
• 反应釜：测试执行时，用例在反应釜中「反应」，显示进度和状态
• 报告墙：右侧为测试报告墙，成功报告贴金箔，失败报告贴红标

测试执行创新：
• 试管动画：每个测试用例放在试管中，执行时液体颜色变化
• 连锁反应：设置测试依赖，前一个成功才触发下一个
• 压力测试模拟：大量并发时显示液体沸腾效果
• 历史对比：可并排对比不同时间点的测试结果
```

### 8. 自动监控&告警页 - 「控制中心」
```
布局概念：航天控制中心
• 大屏墙：主区域为监控大屏，分割显示多个监控视图
• 控制台：弧形控制台在下方面，有物理按钮质感（金色金属）
• 雷达扫描：中间雷达图扫描监控目标，异常目标显示为红点
• 通信频道：右侧显示告警信息流，像太空通信一样滚动

监控可视化创新：
• 3D拓扑图：服务间关系用3D拓扑图显示，节点高度表示负载
• 热成像图：接口响应时间用热成像颜色表示
• 声音可视化：系统日志转换为声波图，异常日志产生「噪声」
• 时间扭曲：可拉伸/压缩时间轴，查看历史监控数据
```

### 9. 部署记录页 - 「时间画廊」
```
布局概念：艺术画廊式时间线
• 画廊长廊：水平无限滚动的时间长廊
• 部署画作：每个部署记录为一张「数字画作」，画框样式表示环境
• 画廊标签：下方时间轴可快速跳转到特定时间段
• 详情展台：点击画作，画作放大到中央展台，显示详细数据

部署可视化创新：
• 部署故事板：每个部署用连环画形式展示各个阶段
• 时间对比：可并排对比两个时间点的部署状态
• 回滚动画：回滚操作显示画作倒放动画
• 签名系统：部署者可在部署记录上留下数字签名
```

### 2. 语音辅助交互
```
语音命令系统：
• 全局命令：「显示项目」「搜索接口」「新建任务」

视觉反馈：
• 语音激活时，界面边缘显示金色脉动波
```

```
操作加速：
```

## 视觉特效系统

### 1. 金色粒子系统
```
应用场景：
• 数据更新时：从更新点散落金色粒子
• 操作成功时：按钮喷射金色粒子
• 加载完成时：粒子汇聚成完成图标
• 错误发生时：红色粒子消散效果

参数控制：
• 粒子数量：5-50个
• 生命周期：0.5-3秒
• 运动轨迹：随机扩散/目标汇聚/螺旋运动
• 消失方式：淡出/缩小/下落
```

### 2. 光线流动系统
```
数据流向可视化：
• API调用：金色光线从请求端流向响应端
• 任务依赖：光线连接依赖任务
• 代码执行：显示代码执行路径光线
• 部署流程：部署各阶段用光线串联

动态效果：
• 流速：表示数据量大小
• 亮度：表示数据重要性
• 频率：表示调用频率
• 颜色：表示数据类型
```

### 3. 玻璃材质系统
```
材质层级：
• 基础玻璃：背景模糊，轻微反射
• 磨砂玻璃：深度模糊，无反射
• 水晶玻璃：清晰折射，高反射
• 金属玻璃：金属质感，镜面反射

应用规则：
• 背景层：基础玻璃
• 内容卡片：磨砂玻璃
• 重要数据：水晶玻璃
• 操作控件：金属玻璃
```

## 自适应布局系统

### 1. 响应式断点
```
超宽屏（>1920px）：
• 显示额外信息列
• 增加侧边栏宽度
• 启用高级可视化

标准屏（1200-1920px）：
• 标准布局
• 完整功能

紧凑屏（900-1200px）：
• 侧边栏自动折叠
• 部分功能移至下拉菜单
• 简化视觉效果
```

### 2. 内容自适应
```
智能内容重组：
• 空间充足时：显示完整图表+详细数据
• 空间中等时：图表简化+关键数据
• 空间紧张时：仅显示数据卡片

动态信息密度：
• 根据用户熟练度调整信息密度
• 新手模式：详细说明+大按钮
• 专家模式：紧凑布局+快捷键提示
```

### 3. 性能自适应
```
硬件感知渲染：
• 高性能设备：启用3D效果、粒子系统、复杂动画
• 中等性能设备：减少粒子数量，简化动画
• 低性能设备：禁用视觉效果，使用静态界面

网络感知加载：
• 高速网络：预加载所有资源
• 中等网络：按需加载
• 低速网络：简化资源，禁用非必要功能
```
